# Verfix API

The Go-based Fiber API layer for the Verfix runtime.

## Role
This service is the primary entry point for the Verfix CLI, SDK, and external AI agents. It handles job ingestion, validates JSON payloads, and enqueues tasks into BullMQ (Redis) for the Node.js workers to execute.

## Key Features
- **High Throughput**: Statically compiled Go binary for minimal latency during enqueue operations.
- **RESTful Endpoints**: Predictable routes for `/verify`, `/executions/:id`, and `/health`.
- **Artifact Serving**: Serves static artifacts (HAR, traces, screenshots) generated by the workers.

## Local Development
```bash
go run main.go
```
