# Verfix Dashboard

The Next.js-based Observability UI.

## Role
Provides human supervisors with deep visibility into the Execution Intelligence Timeline.

## Features
- **Execution Timeline**: Visualizes every step, assertion, and AI reasoning event.
- **Correlated Logs**: Filters network and console logs to match the exact timestamp of timeline events.
- **Metrics**: Tracks flaky assertions and overall reliability.

## Local Development
```bash
npm run dev
```
