# Observability

Autonomous browser execution requires absolute observability. Verfix provides an observability layer that empowers both human supervisors and AI agents.

## Human Observability (The Dashboard)

The Next.js Dashboard visualizes the Execution Timeline:
- **Correlated Logs**: Console and Network logs are filtered to within ±50ms of a specific event in the timeline.
- **Reasoning Visibility**: In Assisted/Exploratory mode, the AI's semantic reasoning is rendered directly in the UI, showing exactly why it chose a fallback selector.
- **Artifacts**: Easy download of HAR files, Playwright Traces, and final screenshots.

## Agent Observability (Structured Outputs)

Agents cannot watch videos. They require structured telemetry.
When an execution fails, Verfix returns:
- A specific failure classification.
- The DOM snippet at the time of failure.
- A semantic hint generated by the runtime regarding why the failure occurred.

This structured observability allows coding agents to self-correct and rewrite tests or code without human intervention.
