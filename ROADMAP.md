# Roadmap

Verfix is continually evolving to provide the most reliable execution infrastructure for AI coding agents.

## Current Focus
- **Execution Observability**: Enhancing the timeline with deeper DOM diffing and network playback.
- **Autonomous Healing**: Improving the semantic recovery rate in Assisted Mode without sacrificing strict determinism.
- **Agent Integrations**: First-class support and native wrappers for LangGraph, CrewAI, and Cursor.

## Upcoming Milestones
- Native WebSockets for real-time execution streaming.
- Cross-browser support (WebKit, Firefox) via Playwright orchestration.
- Execution diffing: Automatically comparing timeline deviations between passing and failing runs.
